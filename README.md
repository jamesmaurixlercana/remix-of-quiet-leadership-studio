# Remix of Quiet Leadership Studio

A Remix.js application inspired by quiet leadership principles.

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will start on http://localhost:3000

### Build

```bash
npm run build
```

This creates an optimized production build in the `dist/` directory.

### Production

```bash
npm start
```

Runs the built application in production mode.

## Project Structure

```
.
├── src/              # Source code
├── dist/             # Build output (generated)
├── package.json      # Project dependencies and scripts
├── tsconfig.json     # TypeScript configuration
├── vite.config.ts    # Vite bundler configuration
└── README.md         # This file
```

## Technology Stack

- **Framework**: Remix.js
- **UI Library**: React 18
- **Language**: TypeScript
- **Build Tool**: Vite
- **Runtime**: Node.js

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
